| CaO Language | Brainfuck |
| ------------ | --------- |
| 艹           | +         |
| 早           | -         |
| 火           | <         |
| 卓           | >         |
| 焯           | .         |
| 草           | ,         |
| ，           |           |
| 。           | ]         |
